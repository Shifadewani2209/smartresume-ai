# SmartResume AI

SmartResume AI is a Generative AI-powered application that analyzes the match between a resume and a job description. It provides intelligent insights to improve resume quality and increase job selection chances.


## 🔥 Features

- 📄 Upload Resume (PDF, DOCX, TXT)
- 📑 Upload Job Description
- 🤖 AI-powered Match Score
- 🔍 Missing Keyword Detection
- ✨ ATS-Optimized Summary Generation
- 🧠 Resume Bullet Improvements
- 🎯 Interview Question Generation
- 📥 Download Analysis as PDF Report


## 🧠 Tech Stack

- Python
- Streamlit
- OpenAI API (LLM)
- PyPDF2
- python-docx
- ReportLab
